**In full bloom** is a **family-owned flower farm** in Moreleta Park, Pretoria, specializing in a wide variety of **South African flowers** 🌻🌹 that are typical in traditional homes, gardens and flower nurseries across the country — from classic **roses** to iconic indigenous **proteas** and **sunflowers**.  

I deliberately chose this type of flower farm because it strongly represents **South African identity**, our rich and unique **flora**, and **cultural heritage**. The farm emphasizes **sustainability**, **smart irrigation systems**, and the transfer of **passion and knowledge** across generations.


## 🎯 Project Goal
To create a **professional**, **visually appealing**, and **fully functional website** that serves as both a **marketing platform** and an **e-commerce/customer service tool** for the flower farm.

## 💡 Skills Highlight
**Technologies & Tools Used:**

- **HTML5** – Used semantic elements such as `<header>`, `<main>`, `<section>`, `<article>`, and `<footer>` to create a well-structured and accessible website. Added proper meta tags, alt text for all images, and meaningful comments throughout the code to make it easier to understand and maintain.

- **CSS** – Built the entire visual design using external CSS files for each page. Applied **Flexbox** and **CSS Grid** for layouts, created smooth hover effects, card flip animations, and custom button styles. Used a consistent color scheme inspired by flowers and made sure the site looks good on all devices.

- **JavaScript** – Added interactivity across the site, including a fully working **shopping cart** 🛒 that saves items with `localStorage`. Implemented dynamic elements like product availability status, toast notifications, form validation, and automatic updates to the cart counter. Used arrays, loops (`forEach`), and conditionals to handle shopping logic and discounts.

- **Responsive Web Design** – Ensured the website works perfectly on mobile phones, tablets, and desktop computers using media queries. The navigation menu stacks vertically on smaller screens, and product grids adjust automatically for better usability on any device. Also included print styles so the pages look clean when printed.

- **Google Fonts & Google Maps Integration** – Integrated **Playfair Display** for elegant headings and **Quicksand** for readable body text. Embedded a **Google Maps iframe** on the Contact Us page to show the exact location of the farm in Moreleta Park, Pretoria.


**Key Skills Demonstrated in short:**
